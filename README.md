# Data Science & Analytics Internship Tasks — DevelopersHub Corporation

## Overview
This repository contains multiple Data Science and Machine Learning tasks completed as part of the DevelopersHub Corporation Internship Program.

The project focuses on applying real-world data science techniques including data cleaning, exploratory data analysis (EDA), visualization, and machine learning model building.

## Objectives
- Perform data cleaning and preprocessing on real datasets  
- Conduct exploratory data analysis (EDA) to extract insights  
- Build machine learning models for classification and regression tasks  
- Evaluate model performance using standard metrics  
- Generate business insights from data  

## Repository Structure
- Task 1: Iris Dataset Analysis (EDA & Visualization)  
- Task 2: Credit Risk Prediction (Classification)  
- Task 3: Customer Churn Prediction (Classification)  
- Task 4: Insurance Claim Prediction (Regression)  
- Task 5: Personal Loan Acceptance Prediction (Classification)  

- datasets/ → All datasets used (.csv, .xlsx)  
- images/ → All generated plots and visualizations  
- notebooks/ → Jupyter notebooks for each task  

## Tools & Technologies
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- Jupyter Notebook / VS Code  

## Key Features
- Data preprocessing and handling missing values  
- Encoding categorical variables  
- Data visualization using graphs and plots  
- Machine learning model training (Logistic Regression, Decision Tree, Linear Regression)  
- Model evaluation using accuracy, confusion matrix, MAE, RMSE  

## Results Summary
- Classification models achieved good accuracy across tasks  
- Key influencing factors identified for each business problem  
- Models successfully used for prediction and decision support  

## How to Run
1. Clone the repository  
2. Install required dependencies:
   pip install -r requirements.txt  
3. Open notebooks in Jupyter or VS Code  
4. Run cells sequentially  

## Conclusion
This project demonstrates practical implementation of data science workflows including data exploration, visualization, and predictive modeling.

Each task focuses on solving a real-world business problem using machine learning techniques.
